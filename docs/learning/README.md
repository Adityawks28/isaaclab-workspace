# Learning artifacts

Self-contained HTML pages the autonomous loop generates to explain *what* was built
and *why* it matters for the RL goal — written to teach, one file per concept or task.

Open any `*.html` here directly in a browser. These pages are **gitignored**
(`/docs/learning/*.html`) and stay private to this machine, like `LEARNING.md`. This
README is the only tracked file in the directory.
