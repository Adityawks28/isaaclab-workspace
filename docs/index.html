Overview

Learn RL by building — the build is the curriculum

Everything runs in Docker (no local install), built FROM the official Isaac Sim image with Isaac Lab + RL libraries layered on top. Custom tasks live in this repo as pip-installable gymnasium plugins, so the stock training and play scripts pick them up without ever editing Isaac Lab. The first milestone is a converged Reach policy; the second carries it into contact-rich pick-and-place — with honest, experiment-driven analysis at every step.

Results & findings

What the experiments actually showed

Controlled studies on the laptop (RTX 4050, 6 GB → 128 envs), fixed seed + budget, compared on weight-independent metrics.

Reach ≈ 0.12 m

  • Converges easily; a reward-weight sweep showed weights set priorities, not absolute quality.
  • Position-tracking error of ~0.12 m end-to-end.

Lift bottleneck found

  • Lift-gate sweep: only the easy gate sustains lifting.
  • More iterations plateau; a goal curriculum doesn't help.
  • Diagnosis: the limiter is grasp reliability, not goal distance — workstation-scale parallelism is the next lever.
